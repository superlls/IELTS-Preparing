# 雅思备考项目指南

## 项目概况
- **目标**: 备考雅思考试
- **主要学习方向**: 听力训练
- **教材**: Cambridge IELTS 官方真题集（5-20+）

## 工作流程

### 学习结构
- 按照 IELTS 真题集逐个 Test 学习
- 每个 Test 包含 4 个 Section
- 针对每个 Section 进行：
  - 初次做题（一刷）并记录成绩
  - 查看答案和得分分析
  - 精听练习（关注听不懂的部分）

### 文件组织
- `Listening/Cambridge IELTS [版本号]/Test[N]/Section[N].md` - 学习笔记和答案
- `Listening/Cambridge IELTS [版本号]/Test[N]/一刷_[日期].md` - 首次做题记录

### 一刷记录格式
创建标题为 `一刷_YYYY-MM-DD` 的文件，包含：
- 各个 Section 的正确率统计
- 平均正确率对比
- 学习计划（重新做题/查看结果/精听练习）

### Section 内容标准结构
每个 Section 文件包含以下部分：
1. **题目** - 原始题目内容
2. **原文** - 完整的听力文本 + 中文翻译
3. **词没听出来的** - 做题时没有直接对应出来的单词/短语列表
4. **错误的地方及原因** - 错题分析和原因说明

### Review Section 生成流程
在一刷记录中为每个 Section 添加 **Review** 部分，包含：

#### 1. 错误分析表
| 错题 | 你的答案 | 正确答案 | 原因分析 |
|-----|--------|--------|--------|
| 列出所有错题 | 你的作答 | 标准答案 | 简要原因（词汇误听/漏听/理解偏差） |

#### 2. 关键词汇梳理
- 对错误涉及的词汇进行展开说明
- 标注 ✓ 掌握的词、❌ 需要强化的词
- 包含同义表达和上下文用法

#### 3. 听力陷阱总结
分类总结本 Section 的典型错误模式：
- 单词混淆（音似但意义不同）
- 词汇匮乏（不认识某个词）
- 漏听现象（词汇在文本中但未捕捉）
- 其他干扰因素（人名地名、语速、口音等）

#### 4. 改进建议
针对错误原因提出具体的学习建议：
- 预热阶段的准备策略
- 词汇积累方向
- 听力技巧优化
- 下次避免的做法

## 建议的学习顺序
1. 选择一个 Test
2. 按 Section 顺序进行一刷，记录成绩
3. 完成所有 Section 后，进行错题分析
4. 进行精听练习，巩固薄弱环节
5. 间隔一段时间后进行二刷/三刷
